Aqui você pode encontrar informações sobre mim e meus projetos.

Este código usa a classe btn da biblioteca Bootstrap para estilizar os botões do menu superior. A classe btn-link define os botões como links sem estilo de botão. Os botões são organizados em uma lista não ordenada dentro da barra de navegação, com a classe navbar-nav. Você pode personalizar os botões do menu superior editando as tags button dentro das tags li da ul com a classe navbar-nav.
